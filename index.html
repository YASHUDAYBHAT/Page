Install Unpacked Version

  1. Download the unpacked BlockMate folder.
  2. Open your browser's extensions page.
  3. Enable Developer Mode (toggle in top right).
  4. Click Load Unpacked and select the downloaded BlockMate folder.
  5. The extension will appear in your browser and be ready to use.
